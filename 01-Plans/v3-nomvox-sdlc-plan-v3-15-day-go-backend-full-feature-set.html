2 · Interaction State Machine

Keyword Intake Form — 8 structured fields
Core idea (required) + 7 optional fields with inline hints. Prompt richness meter turns green at 5+ fields. "Inspire me" auto-fills with an example scenario. Go handler validates and assembles a deterministic LLM prompt — no thin prompts, no wasted regeneration loops.
AI Generation — IBM Granite via watsonx.ai
Produces 8–12 enriched NameCards (streamed). Each card: name, tagline, tone reasoning, origin story, brand score, brand voice samples, short + long descriptions.
80% Availability Gate — Go engine (automatic, invisible to user)
Domain (RDAP) + 5 social handle probes run in parallel. Weighted score applied. Names below 80% silently discarded. Zero-pass → auto-retry Granite up to 3 cycles → closest-match fallback.
User Reaction — three paths
Filtered results shown with score badges and availability matrix.
✓ OPTION 1 — SELECT
Accept a name → move to Visual Identity (Step 3)
✗ OPTION 2 — REJECT
Reject with optional note → AI explains what it heard + asks 1 clarifying Q → session updated → regenerate
↻ OPTION 3 — REPRODUCE
Fresh batch using all accumulated session memory (liked, rejected, direction notes)
Visual Identity — after SELECT
Optional vision image upload seeds all visuals. Mood board (4-panel) · Logo concepts (Profile / App / Business, all regenerable) · Hero landing-page mockup (Granite HTML/CSS) · Brand Persona Card · Brand Voice Samples. Each asset independently regenerable with feedback notes stored in session.
Export Pack — ZIP download
Go archive/zip bundles all assets: names, taglines, brand brief PDF, persona card, voice samples, mood board, 3 logo types (SVG+PNG), hero HTML, color palette JSON.

3 · Innovation Features (Full Set)

Name Origin Story
Each card includes 1–2 sentences on linguistic roots, sound symbolism, and cultural associations. Makes the AI's reasoning visible — exactly what a human naming consultant delivers.
Build cost: zero new infra — one extra Granite instruction per card.
Brand Score Card
Per-name structured score: Memorability (1–10), Spellability (1–10), Global Safety (1–10), Domain Squatter Risk (Low/Med/High). Answers "why this name over that one?" — no tool on the market does this.
Build cost: one structured Granite call per name — no new infra.
Anti-Name Reasoning
On rejection, the AI explains what it heard and asks one clarifying question before regenerating. Transforms the refine loop from a blind spin into a visible conversation. The clearest possible demo of "partner, not generator."
Build cost: one conditional Granite call on reject — modifies existing refine loop.
Brand Persona Card
"Verdara is 28, reads Monocle, uses em-dashes, would never say 'synergy'." After name selection, Granite generates a brand-as-person character card. Goes beyond naming into brand strategy — nothing in this category does this.
Build cost: one Granite call post-selection — zero new infra, maximum judge impact.
Style DNA Slider
Two-axis slider (Playful↔Premium · Abstract↔Descriptive). As the user drags, tonal instructions append to session and next Reproduce reflects them live. Judges will want to play with this in the demo.
Build cost: frontend slider + prompt injection into session — no new API calls.
Brand Voice Samples
For the selected name: an Instagram caption, a welcome email subject line, and a 404 page message — all written in the brand's voice. Proves the AI understands the brand deeply enough to write in it, not just name it.
Build cost: one structured Granite call — included in name-gen call to save a round-trip.
Vision Image Upload
After name selection, user uploads any reference image (photo, screenshot, Pinterest board). Vision model extracts dominant colours, mood adjectives, style keywords, and texture notes. These seed every subsequent image-gen call. Multimodal — shows don't tell.
Build cost: one new Go handler + vision model call. Highest differentiation per build hour.
Competitor Name Radar
After the availability gate, a second Granite pass flags semantic overlap with known brands in the same industry. Shows a "too close to…" warning badge. Saves founders from expensive rebrands — no naming tool currently offers this.
Build cost: one Granite call per batch — added to availability handler response.
