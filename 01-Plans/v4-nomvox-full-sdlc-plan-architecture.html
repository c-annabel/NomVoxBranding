01 Project Overview

NomVox is a conversational AI brand-identity partner that transforms a one-line business concept into a complete, launch-ready brand universe — coined names, cross-platform availability checks, logo concepts, mood boards, brand persona, and a hero landing-page mockup — all in under 60 seconds.

Problem
  • Founders juggle 5+ tabs to name a brand
  • Existing tools (Namelix, Namechk, Looka) are one-shot — no memory
  • "I liked option 3 but want it shorter" → dead end
  • Domain + handle checks are manual and slow
  • Logo costs ≥$500 freelancer or hours of iteration
Solution
  • Coined names — invented vocabulary, never real words
  • Live RDAP + 5 social platform probes per name
  • 60% availability gate filters unusable names
  • Session memory: every like/reject/note feeds next AI call
  • Logo × 3 styles + mood board + landing page mockup
  • Export ZIP with all brand assets
Challenge Fit
  • ✅ AI Creative Partner
  • ✅ Creative Ideation Platform
  • ✅ AI Design & Visual Concept Tools
  • ✅ Personalized Creative Assistant
  • Primary tool: IBM Bob (required)
  • Core AI: IBM Granite / watsonx.ai
Why Web App
  • No App Store review delay
  • Screenshotting/scraping = backend task
  • Faster to demo (screen-record directly)
  • Responsive — works on mobile browser
  • No native capabilities needed (no GPS/camera)

  • RepetitionPenalty=1.05
AI — Images
  • Imagen 3 (Google AI Studio)
  • 4 mood board panels (1:1)
  • 3 logo concepts (parallel)

Session / Data
  • Redis via Upstash (free tier)
  • 2-hour TTL per session
  • Liked / Rejected / Notes / DNA

Availability
  • RDAP (Verisign) — domain
  • Instagram / X / TikTok
  • Threads / YouTube
  • 60% weighted gate
  • Zero-pass → top-2 partials
  • Competitor Radar (Granite)

04 User Flow — Creative Partner Loop

1

Intake Form — 8-field structured prompt builder

Core Idea (required) + 7 optional fields: Target Audience, Brand Personality, Style/Aesthetic, Industry, Colour Mood, Name Length, Avoid. Prompt richness meter shows how much context has been given. Inspire Me button fills all fields with a curated example.

2

AI Name Generation — 4 coined names in ~20s

LLM called with 5-strategy radical naming system prompt + full session context (liked names, rejection notes, slider DNA). Each card includes: Name, Tagline, Origin Story, Brand Score (memorability/spellability/safety/squatter risk), Brand Voice samples (IG/Email/404), Style Tags.

3

Availability Check — parallel probes across 6 platforms

RDAP domain check + Instagram, X, TikTok, Threads, YouTube probed concurrently. Weighted 60% gate filters names. Zero-pass fallback shows top-2 partials with amber warning. Competitor Radar runs semantic clash detection via second LLM call. 60s wall-clock timeout.

4

Creative Partner Loop — Like / Pass (with reason) / Choose

♡ Like → name added to session liked list, influences next regeneration. ✕ Pass (optional reason) → AI fires Anti-Name Reasoning — asks one clarifying question. User answers → injected into session notes → ↻ Regenerate generates fresh names with full context. StyleDNA sliders (Playful↔Premium, Abstract↔Descriptive) shape the next batch.

5

Visual Identity Pack — 3-step sub-flow

Step 1: Logo Concepts — 3 parallel Imagen 3 calls: Geometric Bauhaus profile mark / Gradient Glassmorphism app icon / Horizontal Wordmark business card. User selects one. Step 2: Mood Board — 4-panel Imagen 3 board generated with logo context + brand colours + personality. Brand Persona card ("brand as a person"). Step 3: Landing Page Mockup — Granite generates a complete hero-section HTML with inline CSS, matching brand palette and personality.

6

Export Pack — ZIP downloaded to browser

brand-brief.json (machine-readable), brand-brief.html (NomVox-styled HTML), landing-page.html, logos/logo-profile.png + logo-app-icon.png + logo-business-card.png + selected-logo.png, mood-board/panel-1.png through panel-4.png, README.txt with full brand data + copyright.

08 Risk Register

RiskLikelihoodImpactMitigation
watsonx.ai rate limit during demo Medium Medium IAM token cache (50min) reduces auth calls. Pre-warm with one test call before recording demo. Keep testllm ready to verify connectivity.
Imagen 3 content policy block (logo prompt) Medium Medium CSS placeholder tiles always shown when Imagen unavailable — user can still select a logo direction. Prompts explicitly say "no text, no faces" to avoid policy triggers.
Social platform rate-limits probes (429) Medium Low Unknown probes treated as "taken" (conservative). Amber banner explains "check inconclusive". Zero-pass fallback shows top-2 partials. User can regenerate.
LLM JSON truncation (> 4 names) Low Medium nameCount fixed at 4. max_new_tokens=1800. ExtractJSON repairs truncated arrays. Retry with minimal prompt on first failure.
Redis Upstash free tier limits Low Low Session store is non-fatal — server falls back gracefully when Redis unavailable. Demo uses short sessions. Upgrade to paid tier for production.
Fly.io cold start latency Medium Low Keep Fly.io machine always-on for demo week (min_machines_running=1 in fly.toml). Alternatively, pre-warm with a ping before starting demo recording.

Recording Tips

Before recording

  • Verify /api/ping responds on deployed URL
  • Run one test generation to warm IAM token
  • Set browser zoom to 90% for full card visibility
  • Use OBS Studio or QuickTime for screen capture
  • Pre-fill intake form (don't show typing in video)

During recording

  • Show logo in top-left throughout
  • Narrate what's happening — don't just click
  • Pause on availability badges — judges care about this
  • Emphasise: "AI remembers what I said" on regen
  • Show ZIP contents in file explorer after download
